# Rule Validation Api

A simple rule-validation API

### Technologies

- NodeJS
- Express
- Typescript

### How to run

- Clone this project
- Install nodeJS
- Run npm install in your terminal

#### Commands

- Start: Run npm run dev in your terminal
- Build: Run npm run build in your terminal
- Test: Run npm test in your terminal
